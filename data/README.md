# Data Folder — README

This folder contains all datasets required to replicate the analysis. There are two types of files: those downloaded automatically via API (through `Codigo_datasetv5.py`) and those compiled manually from external sources.

---

## Output Dataset

| File | Description |
|---|---|
| `data_version_auto.csv` | Final hourly dataset (2020–2024) generated by the pipeline. 43,824 rows × 45 columns. All quantities in GW/GWh; prices in €/GWh. |

This is the file consumed directly by the Julia dispatch model.

---

## API-sourced Data (auto-downloaded)

The following data is fetched automatically when running `Codigo_datasetv5.py`. No manual download is needed.

**ESIOS (Red Eléctrica de España)**
- Hourly electricity generation by technology (16 sources)
- Sectoral demand: residential, commercial, industrial
- Hourly spot price (OMIE day-ahead market)
- Net interconnection flows: France, Portugal, Morocco
- Monthly installed capacity by technology (15 sources)
- Self-reported available capacity: hydro, nuclear

**ENTSO-E (European Network of Transmission System Operators)**
- Disaggregated hydroelectric generation: run-of-river, reservoir, pumped storage (generation and consumption)

To run the pipeline you need valid API credentials for both sources. These are set at the top of `Codigo_datasetv5.py`:
```python
TOKEN = "..."          # ESIOS API key
APIKey_entsoe = "..."  # ENTSO-E API key
```

## Manually Compiled Files

The following three files were built manually from public sources and must be present in this folder before running the pipeline. The script will raise a `FileNotFoundError` if any of them is missing.

| File | Frequency | Content | Source |
|---|---|---|---|
| `NaturalGas_Daily_Prices2020-2025.xlsx` | Daily | Natural gas reference price (€/MWh) | MIBGAS |
| `Coal_Diesel_ETS_Monthly_Costs.xlsx` | Monthly | Coal (€/MWh), diesel pre-tax (€/MWh), EU ETS allowance price (€/tCO₂) | World Bank, EEX |
| `Uranium_Annual_Prices2020-2024.xlsx` | Annual | Uranium spot price (€/MWh, converted from USD/lb using a factor of 30.211) | UxC / Cameco |

> **Note:** CSV versions of these files are also accepted by the pipeline. The loader checks for `.xlsx` first and falls back to `.csv` automatically.

---

## How to Reproduce the Dataset

1. Make sure the three manually compiled files above are present in this folder.
2. Set valid API keys in `Codigo_datasetv5.py`.
3. Run the script from the root of the repository:
   ```bash
   python scripts/Codigo_datasetv5.py
   ```
4. The script will write `data_version_auto.csv` to this folder upon completion (~60–90 min depending on API response times).

## Missing Data Treatment

Missing values in the raw API downloads are handled inside the pipeline through a three-step imputation strategy applied in strict order:

1. **1-hour gaps** → forward fill from the immediately preceding hour.
2. **Larger blocks** → values copied from exactly 24 hours prior, preserving diurnal seasonality. *Spot prices are excluded from this step and imputed via linear interpolation instead.*
3. **Remaining gaps** → linear interpolation followed by backward fill for edge cases at the start of the series.

After imputation, all physical variables are clipped to ≥ 0.
